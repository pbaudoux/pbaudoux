# Software
- **[LICEcap](https://www.cockos.com/licecap/)** for creating GIFs screen captures
- **[Rectangle](https://rectangleapp.com/)** for moving, resizing and snapping windows.

# Shortcuts

| Shortcut                  | Result   |
|---------------------------|----------|
| `Option + Shift + ( or )` | [ or ]   |
| `Option + Shift + l`      | \|       |
| `Option + Shift + :`      | \        |
| `Option + n`              | ~        |
